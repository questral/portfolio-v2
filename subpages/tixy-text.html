Tixy Text

Winter 2024

timeframe: 2 weeks

creative code

how can I turn a text field into mathematical playground?

project brief: I challenged myself to create a text focused "shader" based on the website tixy.land

for this project i: developed a system to map mathematical input to color, motion, and typographic content

  • p5js

Tixy Text is a "shader" inspired by the website tixy.land, made using only type. Based on the value returned by a math function I choose (which is always between 0 and 1), each letter will interpolate between two states. It functions as a creative exercise, designed to create thousands of potential permutations based on the colors, patterns, and messages given to it.

Because the characters themselves form part of a message decided by the user, you can see glimpses of coherent words peek through every composition.
