Body Weave

Winter 2024

timeframe: 1 week

creative code

what if you could build your own body?

project brief: create an interactive artwork that takes advantage of the ml5 library within p5js

for this project i: learned to work with a code library, using it to find emergent interactions

  • p5js

Body Weave is an interactive project I made with the help of my professor Golan Levin, with the only requirement being that we use the ml5.js body pose library. A rope connects your two hands, and the user is able to weave it around the tracked points of their body to create new forms for themself. I wanted to explore how our perceptions of self might be altered when our bodies are so heavily abstracted, yet we still maintain some level of control.
